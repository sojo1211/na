# 나루토 기술 목록

> 인식 가능한 인: 쥐·소·호랑이·토끼·용·뱀·말·양·원숭이·닭·개 (11종)
> ❌ 이미지 없음 (제외됨): 멧돼지·합장·임

---

## ✅ 플레이 가능한 기술

| # | 기술명                         | 인 순서                                      | 개수 |
| - | ------------------------------ | -------------------------------------------- | ---- |
| 1 | **분신의 술**            | 양 → 뱀 → 호랑이                           | 3개  |
| 2 | **화둔·용화의 술**      | 뱀 → 용 → 토끼 → 호랑이                   | 4개  |
| 3 | **수둔·수란파의 술**    | 용 → 호랑이 → 토끼                         | 3개  |
| 4 | **소환·토둔 추아의 술** | 호랑이 → 뱀 → 용 → 개                     | 4개  |
| 5 | **화둔·봉선화의 술**    | 쥐 → 호랑이 → 개 → 소 → 토끼 → 호랑이   | 6개  |
| 6 | **수둔·수상어탄의 술**  | 호랑이 → 소 → 용 → 토끼 → 닭 → 용 → 양 | 7개  |
| 7 | **화둔·화룡염탄의 술**  | 양 → 말 → 뱀 → 용 → 쥐 → 소 → 호랑이   | 7개  |
